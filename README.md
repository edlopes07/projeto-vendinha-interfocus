## 📖 Sobre o Projeto

Este projeto consiste em um sistema de controle de vendas e gestão de fiado desenvolvido para uma vendinha local. A solução tem como objetivo facilitar o registro de vendas, o acompanhamento de débitos de clientes e a organização do caixa do estabelecimento.

O sistema foi projetado com foco em simplicidade e acessibilidade, visando atender tanto o proprietário, possivelmente com menor familiaridade com tecnologia, quanto os funcionários no dia a dia.
